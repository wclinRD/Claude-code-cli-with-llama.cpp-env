# Weather Skill

使用 Open-Meteo API 查詢天氣，以繁體中文 Markdown 格式輸出。

## 觸發條件

當用戶輸入包含以下關鍵字時觸發：
- 「天氣」
- 「下雨」
- 「溫度」
- 「氣象」
- 城市/地點名稱 + 時間表達（如「今天竹北天氣」）

## 實作流程

### Step 1: 自然語言解析

從用戶輸入中提取：
1. **地點**：使用正規表達式提取地點名稱
2. **天數**：
   - `今天`/`今日` → 1 天
   - `明天`/`明日` → 1 天
   - `後天` → 2 天
   - `未來N天`/`N天後` → N 天
   - `未來一週`/`這週`/`一週` → 7 天
   - 預設值 → 7 天
3. **意圖**：
   - `比較`/`對比`/`比` → 天氣對比
   - `警告`/`提醒` → 自動天氣警示

### Step 2: 地點解析 (Geocoding API)

使用 Open-Meteo Geocoding API 將地點名稱轉換為經緯度座標。

**API Endpoint:**
```
GET https://geocoding-api.open-meteo.com/v1/search
    ?name={地點}
    &language=zh
    &count=5
    &format=json
```

**搜尋策略（優先查詢台灣）：**

1. **第一優先**：搜尋「{地點} 台灣」（如「竹北 台灣」）
   - 例如：搜尋「竹北 台灣」會直接命中台灣的竹北
2. **第二優先**：搜尋「{地點}市」（如「竹北市」）
   - 適用於「台北」、「台中」等加「市」就能命中
3. **第三優先**：搜尋原詞「{地點}」
4. **最終fallback**：若以上都無結果，從 API 返回的多個結果中優先選擇 `country_code=TW`

**解析邏輯:**
1. 按上述優先順序發送請求，直到取得結果
2. 若有多個結果，優先選擇 `country_code=TW` 的結果
3. 若有多個台灣結果，取 `population` 最高者
4. 回傳 `latitude`, `longitude`, `timezone`, `name`, `country`

**多地點解析:**
- 使用 `、` 或 `和` 或 `,` 分割多個地點
- 支援格式：`竹北、台北天氣`、`竹北和台北的天氣`
- 限制：最多 5 個地點
- 每個地點獨立套用上述搜尋策略（優先台灣）

**模糊匹配:**
- Open-Meteo API 內建 fuzzy matching（3+ 字符）
- 支援常見別名：`台北`/`臺北`、`London`/`Londan`

### Step 3: 天氣查詢 (Weather API)

使用 Open-Meteo Weather Forecast API 取得天氣資料。

**API Endpoint:**
```
GET https://api.open-meteo.com/v1/forecast
    ?latitude={lat}
    &longitude={lon}
    &daily=weather_code,
           temperature_2m_max,
           temperature_2m_min,
           apparent_temperature_max,
           apparent_temperature_min,
           precipitation_sum,
           precipitation_probability_max,
           wind_speed_10m_max,
           uv_index_max,
           sunrise,
           sunset,
           sunshine_duration
    &hourly=temperature_2m,
            precipitation_probability,
            uv_index,
            weather_code
    &current_=temperature_2m,
             relative_humidity_2m,
             apparent_temperature,
             precipitation,
             weather_code,
             wind_speed_10m
    &timezone=auto
    &forecast_days={天數}
```

**參數說明:**
- `daily`: 每日預報資料（最高/最低溫、天氣代碼、降雨機率等）
- `hourly`: 每小時資料（用於時段細分）
- `current`: 當前天氣狀況
- `timezone=auto`: 自動使用當地時區

**天數限制:**
- 最少 1 天
- 最多 16 天

### Step 5: 天氣警示分析

檢查以下條件，自動產生警示：

| 警示類型 | 條件 | 訊息格式 |
|---------|------|---------|
| 🌧️ 降雨提醒 | 降雨機率 ≥ 60% | 明日{地點}降雨機率 {X}%，建議攜帶雨具 |
| ⚠️ 氣溫驟變 | 日溫差 ≥ 10°C | 今日氣溫 {min}°C~{max}°C，溫差 {diff}°C |
| ☀️ 高紫外線 | UV Index ≥ 8 | 今日 UV 指數 {X}（極高），建議塗防曬 |
| 🔥 高溫警示 | 體感溫度 ≥ 35°C | 體感 {X}°C，注意防暑 |
| 🥶 低溫警示 | 體感溫度 ≤ 5°C | 明日體感 {X}°C，注意保暖 |

### Step 7: 進階功能（Phase 2-3）

#### 時段細分 (Phase 3)

根據 `hourly` 資料，分析各時段並生成建議：

```markdown
🕐 **時段建議**
- 🕐 清晨 (05:00-08:00): {溫度}，{天氣}，{建議}
- 🕙 上午 (08:00-12:00): {溫度}，{天氣}，{建議}
- 🌞 中午 (12:00-14:00): {溫度}，{天氣}，{建議}
- 🌧️ 下午 (14:00-18:00): {溫度}，{天氣}，{建議}
- 🌙 晚間 (18:00-22:00): {溫度}，{天氣}，{建議}
```

時段判斷邏輯：
- 05:00-08:00 → 清晨（適合晨間運動）
- 08:00-12:00 → 上午（工作時段）
- 12:00-14:00 → 中午（用餐時段）
- 14:00-18:00 → 下午（午後可能陣雨）
- 18:00-22:00 → 晚間（休閒時段）

#### 天氣對比 (Phase 3)

使用 `past_days` 參數取得歷史資料進行對比：

- **今日 vs 昨日**：使用 `past_days=1`
- **本週 vs 上週**：使用 `past_days=7`

```markdown
📊 **天氣對比**
- 🌡️ 今日最高溫 {X}°C，比昨日 ({Y}°C) **{diff}°C**
- 💧 降雨機率 {X}%，與昨日相同
- 📈 趨勢：明日氣溫 {升/降} {diff}°C
```

#### 穿衣建議 (Phase 3)

根據溫度範圍給出建議：

| 體感溫度 | 建議 |
|---------|------|
| ≥ 35°C | 👕 輕薄衣物，防曬必備，攜帶水瓶防中暑 |
| 28-35°C | 👕 輕薄衣物，適當防曬 |
| 20-28°C | 👕 舒適日常穿著 |
| 15-20°C | 🧥 薄外套，早晚注意保暖 |
| 5-15°C | 🧥 外套必備，注意溫差 |
| < 5°C | 🧥 保暖外套，帽子手套，建議洋蔥式穿法 |

額外建議：
- 降雨機率 > 60% → ☂️ 建議攜帶雨具
- UV Index > 7 → 🧴 高紫外線，務必塗防曬
- 風速 > 30 km/h → 💨 風大，外出注意安全

## 錯誤處理

| 錯誤情境 | 回覆方式 |
|---------|---------|
| 找不到地點 | 抱歉，找不到「{地點}」，請確認名稱是否正確。可嘗試使用更詳細的地址，如「新竹縣竹北市」。 |
| API 失敗 | 無法取得天氣資料，請稍後再試。若問題持續，可能是網路連線問題。 |
| 天數超限 | 抱歉，最長支援 16 天預報。 |
| 多地點過多 | 一次最多查詢 5 個地點，請減少查詢數量。 |

## 實作範例

### 輸入
```
今天竹北天氣怎樣？
```

### 處理流程
1. 解析地點：竹北
2. 解析天數：今天 → 1 天
3. Geocoding API（優先搜尋台灣）：
   - 第一步：搜尋「竹北 台灣」→ 無結果
   - 第二步：搜尋「竹北市」→ ✅ 有結果 (TW)
4. 取得座標：latitude=24.83833, longitude=121.00778
5. Weather API：`https://api.open-meteo.com/v1/forecast?latitude=24.83833&longitude=121.00778&daily=...&timezone=auto&forecast_days=1`
6. 格式化輸出

### 輸出
```markdown
## 🌤️ 竹北市 天氣預報

🔍 查詢地點：竹北市 (新竹縣)

---

### 📅 今日 (2026-04-24)

| 項目 | 數值 |
|------|------|
| 🌤️ 天氣 | ☀️ 晴朗 |
| 🌡️ 溫度 | 22°C ~ 30°C |
| 🌡️ 體感 | 21°C ~ 32°C |
| 💧 降雨機率 | 10% |
| 💨 風速 | 12 km/h |
| 🌊 紫外線 | 極高 (UV 9) |
| 🌅 日出 | 05:32 |
| 🌇 日落 | 18:21 |

👗 穿衣建議：輕薄衣物，防曬必備
```
